# Personal Portfolio Website

This is a basic personal portfolio website made using HTML and CSS. It's a starter project that you can use to showcase your skills, experience, and achievements to potential employers or clients. The website includes an about page, a skills page, an education page, an experience page and an interests page.

Features Responsive design: The website is optimized for different screen sizes, from tablets to desktop computers. Clean and simple layout: The website has a minimalist design with a clear navigation menu and easy-to-read content. CSS: The website includes some basic CSS to add visual interest and make the site more engaging.

How to use To use this project, you can simply download the code or clone the repository to your local machine. Then, you can modify the HTML and CSS files to customize the website to your own needs. You can add your own content, change the colors and fonts, and update the images and links.

Credits This project was inspired by various online tutorials and resources, including W3Schools, Codecademy. The icons used in the website are from Font Awesome.
